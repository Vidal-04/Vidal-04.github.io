More information on Game development

Game Development jobs and what they do:

  1. Computer Programmer
    • Writes code for the game
    • Enhance gameplay and fix issues
    • should have knowledge on multiple coding languages
  2. Game Designer
    • Designs elements of the game
    • Creates models, characters, and plot lines
    • In charge of the aesthetic appeal of the game
  3. Audio Engineer
    • Produces sounds for the game
    • Record and edit voices, sound effects, background music
    • They use all these audios to match the game's theme
